## 📖 关于本站

做笔记的确是一门学问。

搭建这个博客的初衷，是为了拯救我在本地文件夹里日渐混乱的 Markdown 文档。与其让它们在硬盘里发霉，不如整理出来，既是**备忘录**，也是**成长的轨迹**。

在这里，我主张**“纯粹的知识记录”**。我不追求华丽的排版（虽然现在已经挺好看了），更看重内容对未来的自己是否有价值。

主要内容将涵盖：
* **CS Core**: CS61A/B/C, 操作系统, 编译原理等硬核课程。
* **Dev Tech**: VS Code, Git, Linux, Verilog 等工具与语言的踩坑指南。
* **Projects**: 记录从零开始构建项目的完整过程。
